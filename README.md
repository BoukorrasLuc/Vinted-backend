# Vinted-Backend

## About this project

- Back-end project .

I have used these packages :

- Express
- Express
- Formidable
- Mongoose
- Cloudinary
- Dotenv

### USER ROUTES

- POST: /user/signup
- POST: /user/login
- PUT: /user/update/:id
- delete: /user/:id

### PAYMENT ROUTE

- POST: /payment

### OFFER ROUTES

- POST: /offer/publish
- GET: /offer/:id
- PUT: /offer/update/:id
- DELETE: /offer/delete/:id
- GET: /offers (SORT OPTION)
